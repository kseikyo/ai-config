# Interface Strategy Checklist

Print this checklist and use it when planning any interface.

## Layer 1: Experience Foundation

- [ ] **Core Job Statement**: What is the ONE core job?
- [ ] **User Journey**: Mapped step by step?
- [ ] **Friction Points**: All problems identified?
- [ ] **Optimization**: How can we reduce steps/time?
- [ ] **Strategy Document**: Created BEFORE design tools?
- [ ] **Competitive Analysis**: Mapped direct/indirect/aspirational competitors?
- [ ] **Time-to-Value**: How many steps for user to get value?
- [ ] **Friction Sweet Spot**: Too much? Too little? Just right?

## Layer 2: Interface Boosting

- [ ] **50ms Trust Test**: What's the first impression?
- [ ] **Primary CTAs**: Which elements pop?
- [ ] **Secondary Actions**: What fades to background?
- [ ] **Visual Hierarchy**: Does it guide conversion?
- [ ] **Color Strategy**: Strategic or decorative?
- [ ] **First 5 Fundamentals**:
  - [ ] Clean hierarchy
  - [ ] Generous spacing
  - [ ] Professional imagery
  - [ ] Balanced typography
  - [ ] Outcome-focused copy
- [ ] **Modern Tools**:
  - [ ] OKLCH colors (not hex/rgb)
  - [ ] 8pt grid alignment
  - [ ] Border radius 4-8px (not 16px+)
- [ ] **AI Slop Check**:
  - [ ] No extreme border-radius
  - [ ] No mesh gradients
  - [ ] Grid integrity maintained
  - [ ] Intentional friction (progressive disclosure)

## Layer 3: Emotional Design Integration

- [ ] **High-Impact Moments**: Where does emotion matter?
- [ ] **Progress Celebration**: How do we celebrate wins?
- [ ] **Streak System**: What creates return visits?
- [ ] **Achievements**: What keeps them engaged?
- [ ] **Trust Building**: For sensitive audiences?
- [ ] **Win Mapping**: Where are the peaks in our app?
- [ ] **Peak-End Rule**: Asking for upgrades AFTER peak moments?
- [ ] **Emotional Patterns**:
  - [ ] Mascot reactions (distressed when user absent)
  - [ ] Passive-aggressive notifications
  - [ ] Variable rewards
  - [ ] Endowed progress
- [ ] **Balance**: 2 celebrations per guilt trip?
- [ ] **State Machine**: Using level-based responses (1-5)?

---

## Decision Tree

```
Still figuring out the core experience?
├── YES → Focus on Layer 1
│   └── Competitive research first!
└── NO
    └── Experience works but looks generic?
        ├── YES → Focus on Layer 2
        │   └── OKLCH + 8pt grid + professional signals
        └── NO
            └── Need more stickiness?
                └── YES → Focus on Layer 3
                    └── Win map + emotional state machine
```

## Quick Reference

| Layer | Goal | If Skipped |
|-------|------|------------|
| 1: Experience | Speed/friction | Beautiful but unused |
| 2: Interface | Trust | Trusted but not tried |
| 3: Emotional | Attachment | Tried but abandoned |
